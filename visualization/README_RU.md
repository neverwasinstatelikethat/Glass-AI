# Модуль визуализации

## Обзор

Модуль визуализации предоставляет инструменты для создания интерактивных представлений производственных данных и результатов анализа. Этот модуль используется для отображения информации на дашборде оператора и поддержки принятия решений в реальном времени.

## Компоненты

### 1. WebSocket Server (websocket_server.py)
Сервер WebSocket для передачи данных визуализации в реальном времени.

#### Основные возможности:
- **Вещание данных**: Передача потоковых данных визуализации клиентам
- **Управление соединениями**: Обработка множественных WebSocket соединений
- **Синхронизация**: Обеспечение согласованности данных между сервером и клиентами
- **Производительность**: Оптимизированная передача данных с минимальной задержкой

### 2. AR Interface (ar_interface.py)
Интерфейс дополненной реальности для наложения информации на физическое оборудование.

#### Основные возможности:
- **AR наложение**: Отображение данных анализа поверх физического оборудования
- **Интерактивность**: Поддержка жестов и голосовых команд
- **Интеграция с ИИ**: Отображение предсказаний и рекомендаций ИИ
- **Распознавание объектов**: Идентификация оборудования и компонентов

## Использование

### WebSocket сервер:
```python
from visualization.websocket_server import WebSocketVisualizationServer

# Создание сервера визуализации
server = WebSocketVisualizationServer(host="localhost", port=8080)
server.start()

# Отправка данных визуализации
server.broadcast_data({
    "sensor_data": sensor_readings,
    "predictions": ml_predictions,
    "recommendations": rl_recommendations
})
```

### AR интерфейс:
```python
from visualization.ar_interface import ARInterface

# Инициализация AR интерфейса
ar_interface = ARInterface()

# Наложение информации на оборудование
ar_interface.overlay_equipment_info(
    equipment_id="FURNACE_01",
    data={
        "temperature": 1520.5,
        "status": "NORMAL",
        "predictions": {"defect_probability": 0.05}
    }
)
```

## Интеграция

Модуль визуализации тесно интегрирован с другими компонентами системы:
- **Дашборд**: Передача данных для веб-интерфейса оператора
- **Потоковая обработка**: Получение потоковых данных для визуализации
- **ML-модели**: Отображение предсказаний и рекомендаций
- **Цифровой двойник**: Визуализация виртуальной копии производственного процесса

## Преимущества

1. **Реальное время**: Мгновенное отображение изменений в производственном процессе
2. **Интерактивность**: Поддержка различных форм взаимодействия (веб, мобильные устройства, AR)
3. **Интуитивность**: Понятное визуальное представление сложных данных
4. **Гибкость**: Поддержка различных типов визуализации и сценариев
5. **Производительность**: Оптимизированная работа даже при больших объемах данных
