# Модуль физического движка

## Обзор

Модуль физического движка предоставляет математические модели и физические симуляции для понимания и прогнозирования поведения стекла в процессе производства. Этот модуль используется цифровым двойником и симулятором для создания реалистичных моделей производственных процессов.

## Компоненты

### 1. Thermal Dynamics (thermal_dynamics.py)
Модель тепловой динамики для симуляции тепловых процессов в производстве стекла.

#### Основные возможности:
- **Теплопередача**: Моделирование процессов теплопередачи в стекле
- **Температурные градиенты**: Расчет температурных градиентов в различных зонах печи
- **Тепловое расширение**: Учет теплового расширения материалов
- **Энергетический баланс**: Расчет энергетического баланса производственной системы

### 2. Viscosity Model (viscosity_model.py)
Модель вязкости стекла в зависимости от температуры и состава.

#### Основные возможности:
- **Зависимость вязкости от температуры**: Расчет вязкости стекла при различных температурах
- **Влияние состава**: Учет влияния химического состава на вязкость
- **Критические точки**: Определение критических точек вязкости для формования
- **Прогнозирование свойств**: Прогнозирование физических свойств стекла

### 3. Stress Analyzer (stress_analyzer.py)
Анализатор напряжений для определения внутренних напряжений в стекле.

#### Основные возможности:
- **Тепловые напряжения**: Расчет напряжений, вызванных температурными градиентами
- **Механические напряжения**: Анализ напряжений от механических нагрузок
- **Распределение напряжений**: Моделирование распределения напряжений в изделии
- **Прогноз трещин**: Прогнозирование вероятности образования трещин

## Использование

### Тепловая динамика:
```python
from physics_engine.thermal_dynamics import ThermalDynamicsModel

# Создание модели тепловой динамики
thermal_model = ThermalDynamicsModel()

# Расчет температурного профиля
temperature_profile = thermal_model.calculate_temperature_profile(
    furnace_temp=1500,  # °C
    cooling_rate=3.5,   # °C/min
    time_steps=100
)
```

### Модель вязкости:
```python
from physics_engine.viscosity_model import ViscosityModel

# Создание модели вязкости
viscosity_model = ViscosityModel()

# Расчет вязкости при заданной температуре
viscosity = viscosity_model.calculate_viscosity(
    temperature=1200,  # °C
    composition={"SiO2": 0.72, "Na2O": 0.15, "CaO": 0.10, "MgO": 0.03}
)
```

### Анализ напряжений:
```python
from physics_engine.stress_analyzer import StressAnalyzer

# Создание анализатора напряжений
stress_analyzer = StressAnalyzer()

# Расчет тепловых напряжений
thermal_stress = stress_analyzer.calculate_thermal_stress(
    temp_gradient=50,    # °C/mm
    cooling_rate=3.5,    # °C/min
    glass_thickness=10   # mm
)
```

## Интеграция

Модуль физического движка тесно интегрирован с другими компонентами системы:
- **Цифровой двойник**: Используется для создания физически точных моделей
- **Симуляция**: Обеспечивает физическую основу для симулятора производственной линии
- **ML-модели**: Предоставляет физические ограничения и признаки для моделей
- **RL-агент**: Использует физические модели для оценки последствий действий

## Преимущества

1. **Физическая точность**: Реалистичные модели физических процессов производства стекла
2. **Предиктивные возможности**: Возможность прогнозировать поведение стекла при различных условиях
3. **Интеграция с ИИ**: Сочетание физических моделей с машинным обучением
4. **Оптимизация процессов**: Помогает находить оптимальные параметры производства
5. **Предотвращение дефектов**: Позволяет предсказывать и предотвращать образование дефектов
