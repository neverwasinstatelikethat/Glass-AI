# Модуль обучения ML-моделей

## Описание

Этот каталог содержит результаты обучения и метрики производительности для ML-моделей системы предиктивной аналитики производства стекла. Здесь хранятся события TensorBoard, которые фиксируют процесс обучения моделей и их финальные метрики.

## Содержание

- **TensorBoard event logs**: Файлы `events.out.tfevents.*` содержат подробную информацию о процессе обучения моделей, включая:
  - Метрики потерь (loss) на тренировочных и валидационных наборах данных
  - Метрики точности (accuracy), F1-меры, precision и recall для классификационных моделей
  - MSE, MAE, RMSE и R² для регрессионных моделей
  - Графики изменения гиперпараметров во время обучения
  - Временные метки и шаги обучения

## Использование

Для визуализации метрик обучения можно использовать TensorBoard:

```bash
tensorboard --logdir=ml_metrics/tensorboard/training
```

## Назначение

Эти метрики используются для:
1. Мониторинга процесса обучения моделей
2. Сравнения производительности различных архитектур
3. Оптимизации гиперпараметров
4. Выявления переобучения или недообучения
5. Принятия решений о развертывании моделей в продакшен
