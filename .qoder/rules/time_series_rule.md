При работе с временными рядами производства стекла:
- Учитывайте разные частоты сбора данных (данные в реальном времени: 1/сек, критические параметры: 1/мин)
- Обрабатывайте пропуски с помощью линейной интерполяции для краткосрочных пропусков и скользящего среднего для долгосрочных
- Для прогнозирования используйте гибридный подход:
  * Краткосрочные горизонты (15-60 мин): LSTM
  * Среднесрочные (2-6 часов): XGBoost/CatBoost
  * Долгосрочные (8+ часов): Prophet с сезонностью
- Признаки для моделей должны включать:
  * Статистики скользящего окна (mean, std, min, max)
  * Разности во времени (lags)
  * Экспоненциальное сглаживание
  * Сезонные компоненты (час дня, день недели)
- Всегда учитывайте задержки между изменением параметров и появлением дефектов (cross-correlation analysis)
