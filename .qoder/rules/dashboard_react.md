Для React/TypeScript дашборда:
- Используйте TypeScript со строгой типизацией
- Следуйте атомарной архитектуре компонентов
- Для визуализаций используйте:
  * Plotly.js для основных графиков
  * D3.js для кастомных визуализаций и тепловых карт
  * Three.js для 3D-визуализации изделий со стеклом
- Управление состоянием: Redux Toolkit с RTK Query для API вызовов
- Все запросы к бэкенду должны иметь обработку ошибок и загрузочные состояния
- Для real-time обновлений используйте WebSockets
- Интерфейс должен быть полностью адаптивным (Material-UI или Ant Design)
- Цветовая схема для индикаторов качества:
  * Зеленый: >95% (хорошее качество)
  * Желтый: 85-95% (требует внимания)
  * Красный: <85% (критическое качество)
